🚀 AETHERION – Futuristic Corporate Landing Page

An ultra-premium, cinematic corporate landing page featuring a live 3D particle network, glassmorphism architecture, and enterprise-grade motion design. Built with pure HTML, CSS, JavaScript, Canvas API, dhe anime.js.

✨ Features
Immersive 3D Particle Background
Dynamic floating particles with intelligent connections, creating a neural network aesthetic.
Glassmorphism Design System
Frosted glass UI with backdrop blur, neon cyan & purple accents.
Advanced Motion Effects
Staggered animations, scroll reveals, hover transitions (powered by anime.js).
Fully Responsive Architecture
Optimized për mobile, tablet dhe desktop.
Dark / Light Mode
Theme switching me localStorage dhe logo adaptive.
Sticky Shrinking Header
Elegant header që adaptohet gjatë scroll.
Interactive Counters
Animated stats që aktivizohen kur hyjnë në viewport.
Production-Level Code
Strukturë e pastër, semantic HTML, CSS variables dhe JS modular.

🛠️ Technologies Used
HTML5 – Semantic layout
CSS3 – Grid, Flexbox, glassmorphism, variables
JavaScript (ES6+) – DOM, localStorage, Intersection Observer
Canvas API – 2D particle system
anime.js – Animations & counters
Font Awesome 6 – Icons
Google Fonts (Inter) – Typography
🎨 Design Highlights
Color Palette:
#03050b (dark base) + #00e5ff (cyan) + #a855f7 (purple)
Typography:
Inter font – clean, modern, corporate
Effects:
Blur (12–20px), glow shadows, gradient text, hover glow
Interactivity:
Smooth scroll, particle motion, hover lift, FAQ accordion
